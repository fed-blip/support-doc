Не завжди потрібно налаштовувати роутер мануально, для більшості випадків можна завантажити backup з налаштуваннями, але є багато маленьких моментів яких слід дотриувтись. Всі бекапи для різних компаній і різних роутерів я зібрав [в одній папці](https://mintinnovations0-my.sharepoint.com/my?id=%2Fpersonal%2Fdzozulia%5Fmint%2Dinnovations%5Fcom%2FDocuments%2Fsupport%20inst%2Fbackup&viewid=62c09e8d%2D95f8%2D464c%2D9092%2Dda8374827445&login_hint=dzozulia%40mint%2Dinnovations%2Ecom).

При спробі завантажити бекап може вилізти error, ось основні вимоги:

-  Резервна копія має бути створена для тієї ж моделі пристрою.
-  Резервна копія повинна відповідати продуктовому коду та апаратній ревізії пристрою.
-  Обидва пристрої повинні мати однакову версію прошивки.

Потрібно пам'ятати, якщо ви хочете завантажити backup для компанії MediaExpert, то для старих поштоматів не можна використовувати backup з налаштуваннями, тому що на всіх старих поштоматах для сімок використовуються різні PIN коди, і після застосування бекапу, вони зміняться, через це сімка заблокується.

Перевірити продуктовий код роутера можна в RMS, через спеціальні фільтри, також його можна побачити в інтерфесі налаштувань телтоніки.
