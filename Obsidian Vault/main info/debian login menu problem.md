
Іноді виникає проблема коли при ребуті поштомату вилазить вінко логінку в debian:

![[Pasted image 20260113120557.png]]

в такому випадку потрібно спочатку спробувати підключитись по anydesk, якщо буде помилка `desk_rt_ipc_error`, або інша, тоді потрібно буде зробити наступне: 

1) зайти в ui-updater, знайти в списку потрібний поштомат, і зайти в нього
2) потрібно його оновити по цих сценаріях (версія не важлива):
   - dealeateApmBackUp
   - deleateApmVersion
3) зробити ребут пк

Після цього anydesk повинен запрацювати, далі підключаємось і вводимо:

`````bash
df -h /
`````

```bash
sudo apt autoclean
``` 

Потім зновву водимо `df -h` і якщо вільного місця менше 200МБ то звертаємось до розробників. В іншому випадку робимо хард ребут поштомту, проблема повинна зникнути.
Якщо вільної пам'яті залишилось все одно мало, перейдіть в [[clearing Debian memory]].
